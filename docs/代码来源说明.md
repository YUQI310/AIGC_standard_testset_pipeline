# 代码来源说明

本脚本的代码实现基于以下开源工具和库的调用，核心业务规则（许可分流、字段设计、清洗逻辑）由本人根据字节风控合规要求独立设计。

## 主要依赖

- **Hugging Face `datasets`**：下载 HC3 等公开数据集
- **Pillow (PIL)**：图像格式统一转换为 JPEG
- **FFmpeg**：视频格式统一转换为 MP4（H.264/AAC）
- **Python 标准库**：json、hashlib、logging、os、sys
